DOM: Документная модель объекта

Представим HTML как дерево:

  1. Корень дерева — элемент <html>.
  2. Ветки — элементы HTML (например, <head>, <body>, <div>, <p>, <img> и т.д.).
  3. Листья — текстовые узлы (текст внутри элементов).

DOM превращает это дерево в иерархическую структуру объектов:

  1. Каждый элемент HTML представлен в виде объекта с собственными свойствами и методами.
  2. Свойства объектов DOM хранят информацию об элементе, например, его имя тега, атрибуты и текстовое содержание.
  3. Методы объектов DOM позволяют изменять элемент, например, добавлять или удалять элементы, изменять их стили, атрибуты и текст.

Как используется DOM:

  1. Доступ к элементам HTML: JavaScript может получить доступ к любому элементу HTML через DOM.
  2. Изменение содержимого страницы: С помощью DOM можно добавлять, удалять и изменять элементы HTML.
  3. Изменение стилей элементов: JavaScript может изменять CSS-стили элементов через DOM.
  4. Обработка событий: DOM позволяет обрабатывать события пользователя, такие как нажатие на кнопку, наведение курсора и т.д.

Примеры использования DOM:

  1. Изменение текста элемента:

        document.getElementById("change-button").addEventListener("click", function() {
        document.getElementById("text-to-change").innerHTML = "Текст изменен!";
    });

        document.getElementById("add-item-button").addEventListener("click", function() {
