Свойство firstElementChild

Свойство firstElementChild в JavaScript позволяет получить ссылку на первый дочерний элемент (элемент внутри) указанного HTML-элемента.

Пример:

Представьте, что у вас есть следующий HTML-код:

        <div id="myDiv">
        <p>Первый  параграф</p>
        <span>Второй  элемент</span>
        <p>Третий  параграф</p>
        </div>

В этом примере div с идентификатором myDiv имеет три дочерних элемента: два параграфа (p) и один элемент span.

Использование firstElementChild

        const showFirstChildButton = document.getElementById("showFirstChild");
        const resultParagraph = document.getElementById("result"); 
        showFirstChildButton.addEventListener("click", function() {
            const myDiv = document.getElementById("myDiv");
            const firstChild = myDiv.firstElementChild; 
            resultParagraph.textContent = `Первый дочерний элемент: ${firstChild.textContent}`; // Выводим текст первого дочернего элемента
        });

В этом коде:

Находим элементы:

  • document.getElementById() ищет элементы HTML по их ID (идентификаторам).
  • showFirstChildButton - это кнопка.
  • resultParagraph - это параграф, куда будет выводиться информация.
  • myDiv - это контейнер (div), содержащий другие элементы.

Обрабатываем нажатие на кнопку:

  • addEventListener("click", function() { ... }); прикрепляет функцию к событию нажатия на кнопку.
  • Когда пользователь кликает по кнопке, функция запускается.

Получаем информацию:

  • firstChild = myDiv.firstElementChild; получает первый дочерний элемент myDiv.
  • firstChild.textContent получает текст из этого первого дочернего элемента.

Выводим информацию:

  • resultParagraph.textContent = ...; устанавливает новый текст в параграф resultParagraph, используя текст, который был получен из firstChild.

Важно отметить:

  1. firstElementChild возвращает null, если у элемента нет дочерних элементов.

  2. firstElementChild возвращает ссылку на первый дочерний элемент, а не текстовое содержание элемента.

Примеры использования firstElementChild:

  1. Изменение стиля первого дочернего элемента:

            const changeColorButton = document.getElementById("changeColor");
        changeColorButton.addEventListener("click", function() {
            const myDivv = document.getElementById("myDivv");
            const firstChild = myDivv.firstElementChild; 
            firstChild.style.color = "blue"; 
        });

Использование firstElementChild позволяет управлять первым дочерним элементом HTML-элемента с помощью JavaScript, что делает веб-страницы более динамичными и интерактивными.
